### Introduction:

*file2lowercase* - convert filename to lowercase.

### Dependencies:

This script makes use of the following python modules:

1. os
2. click

### Usage:

Parameters supported by this script include:

| Option               | Description                             |
|:---------------------|:----------------------------------------|
| `FILENAME`           | File[s] to be renamed. |
| `-f`                 | Force overwritting files. |
| `-v`                 | Enable verbose mode. |
| `--help`             | Show help message and exit.             |
